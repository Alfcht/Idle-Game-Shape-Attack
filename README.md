# Idle Game: Shape Attack

Small Unity idle/tower-like game — shapes attack and towers defend.

Quick start

- Open the project folder in Unity (use a compatible Unity Editor version).
- Open the scene: `Assets/Scenes/MainMenu.unity` and press Play.

Repo contents

- `Assets/` — game assets and scripts (source)
- `ProjectSettings/` — Unity project settings
- `Packages/` — manifest for package dependencies

Notes

- Do not commit `Library/`, `Temp/`, `obj/` or other generated folders — they are excluded via `.gitignore`.
- If you see CRLF/LF warnings, add a `.gitattributes` (already included) to normalize line endings.

Contributing

- Create a branch for your change, add tests where applicable, and open a Pull Request.
